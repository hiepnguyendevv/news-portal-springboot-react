========================================
DANH SÁCH CHỨC NĂNG HIỆN TẠI
========================================

1. AUTHENTICATION & AUTHORIZATION
   - Đăng ký tài khoản
   - Đăng nhập (Username/Password)
   - Đăng nhập Google (OAuth2)
   - Refresh Token tự động
   - Đăng xuất
   - Xem/Cập nhật profile
   - Restore session khi F5 trang

2. QUẢN LÝ TIN TỨC
   - Xem danh sách tin (phân trang)
   - Xem chi tiết tin
   - Tìm kiếm tin
   - Lọc theo category (theo tên và slug)
   - Sắp xếp theo lượt xem (tăng/giảm)
   - Đếm lượt xem (tracking visitor data)
   - User: Tạo/Sửa/Xóa/Gửi duyệt tin
   - Admin: Duyệt/Từ chối/Quản lý tin
   - Xóa/Duyệt hàng loạt
   - Upload ảnh đại diện cho tin tức

3. LIVE NEWS (REAL-TIME)
   - Tạo Live News (chỉ Admin)
   - Thêm/Cập nhật/Xóa entries
   - Real-time updates qua WebSocket
   - Upload media cho entries
   - Xem Live News với phân trang
   - Sắp xếp entries theo thời gian

4. COMMENTS
   - Xem comments (nhiều cấp - nested)
   - Thêm/Xóa comment
   - Like/Unlike comment
   - Reply comment (trả lời comment)
   - Admin: Quản lý, tìm kiếm, xóa/khôi phục
   - Soft delete comments

5. BOOKMARKS
   - Lưu/Bỏ lưu bài viết
   - Xem danh sách đã lưu (phân trang)
   - Kiểm tra trạng thái bookmark

6. CATEGORIES
   - Xem categories (hierarchical - có parent/child)
   - Xem subcategories
   - Admin: Tạo/Sửa/Xóa category
   - Xóa hàng loạt
   - Đếm số bài viết theo category

7. TAGS
   - Auto-generate tags từ nội dung
   - Xem tất cả tags
   - Admin: Tạo/Sửa/Xóa tag
   - Hiển thị tags trên bài viết

8. NOTIFICATIONS
   - Xem thông báo
   - Đếm chưa đọc
   - Đánh dấu đã đọc

9. USER MANAGEMENT (ADMIN)
   - Xem/Tạo/Sửa/Xóa user
   - Cập nhật status (ACTIVE/INACTIVE)
   - Xóa/Cập nhật hàng loạt
   - Soft delete user

10. MEDIA
    - Upload ảnh/video
    - Tích hợp Cloudinary
    - Upload ảnh cho tin tức

11. SECURITY
    - JWT Authentication
    - Refresh Token (HttpOnly cookie)
    - Token rotation
    - Role-based access (USER/ADMIN)
    - Password BCrypt encryption
    - WebSocket authentication

12. FRONTEND
    - Responsive UI
    - Protected routes
    - Admin routes
    - Guest routes
    - Toast notifications
    - Pagination
    - Loading states
    - Error handling

========================================
TỔNG KẾT
========================================
- API Endpoints: ~75+
- Controllers: 14
- Frontend Pages: 20+
- Database: MySQL + Redis
- Real-time: WebSocket + Redis Pub/Sub
- Media: Cloudinary integration

========================================
ĐỀ XUẤT CHỨC NĂNG MỚI
========================================

1. TÍNH NĂNG TƯƠNG TÁC & XÃ HỘI
   ⭐ Like/Dislike bài viết
      - Cho phép user like/dislike bài viết
      - Hiển thị số lượt like
      - Lưu lịch sử like của user
   
   ⭐ Share bài viết
      - Share lên Facebook, Twitter, LinkedIn
      - Copy link bài viết
      - Tạo short URL
      - Đếm số lượt share
   
   ⭐ Follow/Unfollow tác giả
      - Follow các tác giả yêu thích
      - Xem danh sách bài viết của tác giả đã follow
      - Thông báo khi tác giả đăng bài mới
   
   ⭐ Rating bài viết
      - Đánh giá bài viết theo sao (1-5 sao)
      - Hiển thị điểm trung bình
      - Xem đánh giá của người khác

2. TÍNH NĂNG TÌM KIẾM & LỌC NÂNG CAO
   ⭐ Tìm kiếm nâng cao
      - Tìm theo tag
      - Tìm theo tác giả
      - Tìm theo khoảng thời gian
      - Tìm theo lượt xem/like
      - Lọc kết hợp nhiều điều kiện
   
   ⭐ Trending/Hot news
      - Tính toán trending dựa trên view, like, comment trong 24h
      - Hiển thị "Hot" badge
      - Trang Trending riêng
   
   ⭐ Lọc theo thời gian
      - Hôm nay, tuần này, tháng này
      - Tùy chọn khoảng thời gian
   
   ⭐ Gợi ý bài viết liên quan
      - Dựa trên category, tags
      - Machine learning recommendations
      - "Bạn có thể quan tâm"

3. TÍNH NĂNG CÁ NHÂN HÓA
   ⭐ Reading history
      - Lưu lịch sử đọc bài
      - Xem lại các bài đã đọc
      - Đánh dấu "Đã đọc"
      - Xóa lịch sử
   
   ⭐ Reading preferences
      - Chọn categories yêu thích
      - Chọn tác giả yêu thích
      - Tùy chỉnh trang chủ
   
   ⭐ Reading list/Collections
      - Tạo danh sách đọc tùy chỉnh
      - Thêm bài viết vào collection
      - Chia sẻ collection
   
   ⭐ Dark mode
      - Chuyển đổi dark/light theme
      - Lưu preference

4. TÍNH NĂNG THÔNG BÁO & TƯƠNG TÁC
   ⭐ Push notifications
      - Browser push notifications
      - Thông báo bài viết mới từ tác giả follow
      - Thông báo reply comment
      - Thông báo bài viết được duyệt/từ chối
   
   ⭐ Email notifications
      - Gửi email khi có thông báo quan trọng
      - Newsletter hàng tuần
      - Digest email
   
   ⭐ Real-time notifications
      - WebSocket cho notifications
      - Badge số thông báo chưa đọc
      - Dropdown notifications

5. TÍNH NĂNG NỘI DUNG
   ⭐ Draft auto-save
      - Tự động lưu nháp khi viết bài
      - Khôi phục nháp
      - Lịch sử chỉnh sửa
   
   ⭐ Rich text editor nâng cao
      - Thêm video embed (YouTube, Vimeo)
      - Thêm code blocks với syntax highlighting
      - Thêm tables
      - Thêm quotes
   
   ⭐ Featured/Highlighted news
      - Admin đánh dấu bài nổi bật
      - Hiển thị ở đầu trang chủ
      - Carousel slider
   
   ⭐ News scheduling
      - Lên lịch đăng bài
      - Auto-publish theo thời gian
   
   ⭐ News versioning
      - Lưu các phiên bản bài viết
      - So sánh các phiên bản
      - Rollback về phiên bản cũ

6. TÍNH NĂNG PHÂN TÍCH & THỐNG KÊ
   ⭐ Analytics dashboard (Admin)
      - Tổng số bài viết, users, comments
      - Biểu đồ thống kê theo thời gian
      - Top bài viết xem nhiều nhất
      - Top tác giả
      - Thống kê theo category
   
   ⭐ Author analytics
      - Số lượt xem bài viết của mình
      - Số lượt like, comment
      - Biểu đồ performance
   
   ⭐ View analytics
      - Xem chi tiết lượt xem (theo ngày, giờ)
      - Xem theo địa lý (nếu có IP tracking)
      - Xem theo device/browser

7. TÍNH NĂNG BẢO MẬT & MODERATION
